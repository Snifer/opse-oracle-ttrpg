## What is OPSE? / ¿Qué es OPSE?

[One Page Solo Engine](https://inflatablestudios.itch.io/one-page-solo-engine) is a minimal, all-in-one GM emulator for solo tabletop RPGs, created by Karl Hendricks (Inflatable Studios).
It uses a single page of tables, a d6 and a deck of playing cards to answer oracle questions, generate scene complications, random events, hooks and NPCs.

[One Page Solo Engine](https://inflatablestudios.itch.io/one-page-solo-engine) es un motor de juego en solitario minimalista y todo en uno, creado por Karl Hendricks (Inflatable Studios).
Con una sola hoja de tablas, un d6 y una baraja de naipes, te permite emular a un DJ: hacer preguntas al oráculo, generar complicaciones de escena, eventos aleatorios, ganchos y PNJ.

This Obsidian plugin brings OPSE tools directly into your vault, so you can play and journal your solo campaigns in one place.  
Este plugin lleva las herramientas de OPSE directamente a tu bóveda de Obsidian, para que puedas jugar y registrar tus campañas en solitario en el mismo sitio.

## Features / Funcionalidades

- Oracle actions for Yes/No questions with Focus tables.
- Scene setup helpers: scene complications and GM moves.
- Random Event, Plot Hook and NPC generators.
- Optional dungeon and hex exploration trackers.
- Command palette integration and configurable hotkeys.

- Acciones de Oráculo para preguntas Sí/No con tablas de Enfoque.
- Ayuda para Preparar la Escena y generar Complicaciones de Escena y Movimientos de DJ.
- Generadores de Evento Aleatorio, Gancho Argumental y PNJ.
- Trackers opcionales para mazmorra y hex-crawl.
- Comandos en la paleta y atajos de teclado configurables.

## Installation / Instalación

### Via BRAT (recommended) / Vía BRAT (recomendado)

1. Install the **BRAT** plugin in Obsidian.
2. Add this repository URL to BRAT’s “Add beta plugin” list.
3. Enable **OPSE for Obsidian** in Settings → Community plugins.

1. Instala el plugin **BRAT** en Obsidian.
2. Añade la URL de este repositorio en la sección “Add beta plugin”.
3. Activa **OPSE para Obsidian** en Ajustes → Plugins de comunidad.

### Manual installation / Instalación manual

1. Download the latest release from the Releases page.
2. Extract the folder `obsidian-opse-plugin` into your vault under `.obsidian/plugins/`.
3. Reload Obsidian and enable **OPSE for Obsidian**.

1. Descarga la última versión desde la sección Releases.
2. Copia la carpeta `obsidian-opse-plugin` dentro de `.obsidian/plugins/` en tu bóveda.
3. Recarga Obsidian y habilita **OPSE para Obsidian**.

## Usage / Uso básico

- Open the command palette and run **OPSE: Ask the Oracle** to insert a Yes/No oracle result into the current note.
- Use **OPSE: Set the Scene** to roll a scene complication and describe where your character is and what they are trying to accomplish.
- Use **OPSE: Random Event** or **OPSE: Plot Hook + NPC** to introduce new twists to your solo campaign.
- Optional side views let you track scenes, threads, dungeons and hex crawls.

- Abre la paleta de comandos y ejecuta **OPSE: Preguntar al Oráculo** para insertar un resultado de Sí/No en la nota activa.
- Usa **OPSE: Preparar la Escena** para tirar una complicación de escena y describir dónde estás y qué intentas conseguir.
- Ejecuta **OPSE: Evento Aleatorio** o **OPSE: Gancho + PNJ** para añadir giros a tu campaña en solitario.
- (Opcional) Abre las vistas laterales para llevar el registro de escenas, hilos, mazmorras y hexágonos.
